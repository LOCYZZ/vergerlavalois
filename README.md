## **1. Introduction**
Ce site est une **réalisation partielle** d'une maquette de site web en **HTML et CSS**, conçue pour être fonctionnelle sur **PC**. Il a pour objectif de présenter un **verger lavallois fictif**, en mettant en avant son histoire, ses produits et ses valeurs.

---

## **2. Structure du Projet**

### **Fichiers Principaux**
- **`index.html`** : Page d’accueil (présentation générale, produits, histoire, etc.).
- **`story.html`** : Page "Histoire" du verger.
- **`produits.html`** : Page listant les produits dérivés (jus, compotes, confitures).
- **`produit1.html`** : Exemple de page détaillée pour un produit (ex. : Jus de pomme).
- **`style.css`** : Feuille de style globale pour tout le site.

### **Dossiers**
- **`assets/`** : Contient toutes les **images et icônes** utilisées sur le site.

---

## **3. Styles CSS**

### **Couleurs Principales**
| Couleur       | Code Hexadécimal | Utilisation                     |
|---------------|------------------|---------------------------------|
| Vert clair    | `#97D109`        | Header, boutons, accents       |
| Marron        | `#A6653F`        | Boîtes de contenu, fonds        |
| Vert foncé    | `#47C75A`        | Boutons de produits            |
| Jaune         | `#FEC851`        | Effets de survol                |

### **Polices**
- **Titres** : `'Cherry Bomb One'` (via Google Fonts).
- **Texte** : `'Raleway'` (via Google Fonts), avec différentes graisses pour les titres et le corps du texte.

---

## **4. Fonctionnalités Disponibles**
- **Page d'accueil** : Présentation générale du verger.
- **Page d'histoire** : Récit détaillé de la création du verger.
- **Page de présentation des produits** : Liste des produits dérivés (jus, compotes, confitures).
- **Page détaillée du jus de pomme** : Exemple de page produit.

> **Note** : Le reste des fonctionnalités (comme les pages détaillées pour les autres produits ou une version mobile) n'a pas été réalisé, car non demandé initialement.
